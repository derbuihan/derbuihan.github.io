Proof-Generalを用いたCoqの環境構築

 2018-12-20  601 字  Coq Emacs 環境構築 

Mac編

brewとbrew caskを用いた環境構築を行います。

GUI版Emacsのインストール

brew cask install emacs

Coqのインストール

brewでCoqをインストールします。

brew install coq

Emacsの設定

最近のEmacsはパッケージマネージャも付属しているので、それを用いてProof-Generalもインストールします。

(require 'package)
(let* ((no-ssl (and (memq system-type '(windows-nt ms-dos))
                    (not (gnutls-available-p))))
       (proto (if no-ssl "http" "https")))
  (add-to-list 'package-archives
               (cons "melpa" (concat proto "://melpa.org/packages/")) t))
(package-initialize)

を~/.emacs.d/init.elに追加する。 これでemacsでpackage managerを使うことができる。

emacsを起動して

  1. M-x package-refresh-contents RET ←初回のみ実行
  2. M-x Package-install RET proof-general RET

でproof-generalをインストールすることができる。

brewを用いてインストールして、No such file or directory, coqtopと出た時は、

(custom-set-variables '(coq-prog-name "/usr/local/bin/coqtop") '(proof-three-window-enable t))

を~/.emacs.d/init.elを追加すれば解決する。

補完のパッケージの追加

company-coqは最強なのでこれも入れましょう。 emacsを起動して、

  1. M-x Package-install RET company-coq RET

を実行。

~/.emacs.d/init.elに

(add-hook 'coq-mode-hook #'company-coq-mode)

を追加すると良い。

ssreflectのインストール

brew install ssreflect

ProofGeneralの使い方

ProofGeneralで使うキーバインド

キー 動作
C-c C-n 次の.まで進む
C-c C-u 次の.まで進む
C-c C-Enter カーソル位置まで進める
